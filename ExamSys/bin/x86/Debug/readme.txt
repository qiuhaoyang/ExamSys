本软件需要.net 4.0的支持，win7 32位系统部分无法安装.net 4.0，请按以下步骤进行操作：
1.按组合键win+R，打开运行，输入cmd，回车，在输入net stop wuauserv，回车，即停止了update的服务；
2.打开C盘根目录下的“Windows”文件夹，找到SoftwareDistribution文件夹，将其重命名为SDold；
3.按组合键win+R，打开运行，输入cmd，回车，在输入net start wuauserv，回车，即启动了update的服务；
4.现在安装.net framework 4.0就会成功了！

软件使用方法：
1.将试题库按本软件自带Excel的格式进行录入，其中sheet1为单选题，sheet2为多选题；
    注意：不要修改文件的表单名称，即默认为sheet1和sheet2
2.双击ExamSYS.exe运行软件，第一次打开软件时，需点击“考试管理”，点击“考试设置”，进行各种题型数量分配设置。设置完毕后，若有设置调整，则重新进行设置，否则无须再次设置；
3.点击“开始考试”，输入姓名、证件号，选择单位名称，即开始考试；
4.考试完毕后，自动报出所得分数，并将分数保存在Score.txt中；
5.点击“模拟考试”，则无须输入个人信息即可进行模拟考试，每做一题，将提示正确答案，全部完成后将自动报出所得分数，但不保存；
6.进行新一轮考试时，建议点击“考试管理”，点击“清空成绩”，将之前保存的考试成绩清空。
